Bienvenidos - Programa de agendamiento de visitas





Programa en Java que genera agendamientos de visitas, de acuerdo a parámetros establecidos para la asignación de cada visita tales como: fecha, hora, lugar de la visita, motivo de la visita y técnico asignado, el programa debe tener las opciones de crear, modificar, cancelar e iniciar las visitas para llevar los registros a través de tablas almacenadas en un sistema de gestión de bases de datos SQL.

Para acceder a la documentación, copia y pega este URL en el navegador: https://github.com/GatoCabanzo/Repo_Ing_de_Software/wiki
